# Dilly Dally Coffee Co.

Speculative redesign by NeonFrame Web Design for Dilly Dally Coffee Co., a coffee shop in Bend, OR named after the Dilly Dally Alley run at Mt. Bachelor.

## Stack

- Pure HTML, CSS, vanilla JS (no build step)
- GSAP 3.12 + ScrollTrigger (CDN, deferred, desktop only)
- Lenis smooth scroll (CDN, deferred, desktop only)
- Google Fonts: Fraunces, Inter, JetBrains Mono, Caveat via non-blocking load
- Self-hosted JPGs sourced from @dillydallycoffeeco Instagram and Avants event listings (see ATTRIBUTIONS.md)

## Pages

- `index.html` Home: hero, origin story, menu teaser, Cars & Coffee feature, Thump partnership, visit block
- `menu.html` Full menu with categories
- `cars-and-coffee.html` Cars & Coffee schedule, rules, photo, RSVP mailto
- `404.html` Page overboard

## Local development

```bash
python3 -m http.server 8765
# open http://localhost:8765
```

## Copy rules

- No em-dashes, no en-dashes, no HTML entities like `&mdash;` or `&ndash;`
- Warm, confident, second-person
- Honor the origin: Jack, Mt. Bachelor, Dilly Dally Alley, Thump, by name
